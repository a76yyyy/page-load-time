# 文档目录

本目录包含 Page Load Time 扩展的技术文档。

## 文档列表

### [architecture.md](./architecture.md)

**架构设计文档**

包含:

- 核心组件说明
- 关键设计决策及其原因
- 数据流和生命周期管理
- 性能优化策略
- 跨浏览器兼容性方案
- **IndexedDB 存储架构和自动清理机制**
- **错误处理方案（QuotaExceededError、InvalidStateError、TypeError）**
- **日志系统设计（级别、格式、模块标识）**

**适合**: 了解系统整体设计、理解技术选型、理解存储和错误处理、日志规范

### [development.md](./development.md)

**开发指南**

包含:

- 环境设置和项目结构
- 调试方法和工具
- **IndexedDB 调试命令**
- **日志说明（级别、格式、过滤）**
- 常见问题排查
- 测试场景和方法
- 代码规范和最佳实践（包括日志规范）

**适合**: 日常开发、调试问题、贡献代码、理解日志输出

## 快速开始

1. **新手入门**: 先阅读 `architecture.md` 了解整体设计
2. **开始开发**: 参考 `development.md` 设置环境和调试
3. **遇到问题**: 查看 `development.md` 的常见问题章节或 `architecture.md` 的错误处理章节

## 文档维护原则

- ✅ **简洁**: 只保留最终方案,不记录探索过程
- ✅ **实用**: 聚焦于"是什么"和"怎么做",而非"为什么这样演变"
- ✅ **同步**: 代码变更时及时更新文档
- ✅ **清晰**: 使用图表、代码示例和清单
- ✅ **整合**: 相关内容整合到主文档中，避免重复
